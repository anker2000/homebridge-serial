# Homebridge Serial

## Introduction
Welcome to the Serial Mini and Serial Pro plugin for [Homebridge](https://github.com/nfarina/homebridge).

This plugin allows you to control your RM Mini and RM Pro with HomeKit using the Home app and Siri.
